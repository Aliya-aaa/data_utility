### Задание 5
Нужно реализовать консольную программу «Утилита обработки данных».
#### Алгоритм:
1. открыть файл
2. читать построчно
3. разделить строку
4. проверить: пустые значения, id — число, value — число
5. если ок → valid_records, иначе → invalid_records + ошибка
6. посчитать статистику
#### Формат входных строк:
В каждой строке должны быть три поля разделенных точкой с запятой (;):
1. id: int
2. name: str
3. value: float
#### Запуск программы:
1. Открыть папку проекта в терминале
2. Проверить установлен ли питон: 
```
python --version
```
3. Запустите программу:
```
python main.py input.txt output.txt
```
где, input.txt - файл с входными данными, output.txt - файл, где сохранен репорт.
Программа выведет результат в терминал и сохрант его в файл.
